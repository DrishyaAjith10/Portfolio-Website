# Drishya Ajith - Interactive Portfolio Website

A stunning, modern, and highly interactive portfolio website showcasing Drishya Ajith's technical skills, projects, and achievements. Built with HTML5, CSS3, and vanilla JavaScript with smooth animations and responsive design.

> **Note**: This portfolio was created with the assistance of Claude AI, demonstrating the collaborative potential of human creativity and AI technology in web development.

## 🌟 Features

### ✨ Interactive Elements
- **Typing Animation**: Dynamic text animation in the hero section
- **Particle System**: Floating geometric shapes in the background
- **Custom Cursor**: Interactive cursor with hover effects
- **Smooth Scrolling**: Seamless navigation between sections
- **Scroll Animations**: Elements animate as they come into view
- **Project Modals**: Interactive project showcase with detailed information

### 🎨 Design Features
- **Modern UI**: Clean, minimal design with vibrant accents
- **Color Scheme**: Light background with coral, teal, and indigo accents
- **Typography**: Inter and Poppins fonts from Google Fonts
- **Glassmorphism**: Subtle frosted glass effects
- **Responsive Design**: Fully responsive across all devices
- **Dark/Light Mode Ready**: Easy theme switching capability

### 📱 Responsive Sections
- **Hero Section**: Animated introduction with particle background
- **About**: Personal information with animated statistics
- **Education**: Timeline-based academic journey
- **Experience**: Interactive cards with hover effects
- **Skills**: Animated progress bars and skill visualization
- **Projects**: Interactive project showcase with modals
- **Leadership**: Leadership experience and achievements
- **Contact**: Functional contact form with validation

### 🚀 Performance Features
- **Preloader**: Creative loading animation
- **Scroll Progress**: Visual progress indicator
- **Back to Top**: Smooth return to top functionality
- **Lazy Loading**: Optimized image loading
- **Throttled Events**: Performance-optimized scroll handlers

## 🛠️ Technologies Used

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone or Download**
   ```bash
   git clone <repository-url>
   cd Portfolio
   ```

2. **Open in Browser**
   - Double-click `index.html` to open directly in browser
   - Or use a local server for better development experience

3. **Using Live Server (VS Code)**
   - Install Live Server extension
   - Right-click `index.html` → "Open with Live Server"

## 🎯 Key Features Explained

### Typing Animation
The hero section features a dynamic typing animation that cycles through different roles and descriptions, creating an engaging first impression.

### Particle System
A custom particle system creates floating geometric shapes in the background, adding visual interest without being distracting.

### Interactive Skills
Skills are displayed with animated progress bars that fill up as they come into view, providing a visual representation of proficiency levels.

### Project Showcase
Projects are presented in interactive cards that open detailed modals with comprehensive information, tech stacks, and links.

### Contact Form
A fully functional contact form with client-side validation and success notifications.

## 🎨 Customization

### Colors
The color scheme can be easily modified by updating CSS variables in `style.css`:

```css
:root {
    --accent-coral: #ff6b6b;
    --accent-teal: #4ecdc4;
    --accent-indigo: #6366f1;
    --text-charcoal: #2d3748;
}
```

### Content
Update the content in `index.html` to personalize:
- Personal information
- Education details
- Experience and achievements
- Skills and proficiency levels
- Project information
- Contact details

### Animations
Modify animation timing and effects in `style.css` and `script.js` to match your preferences.
